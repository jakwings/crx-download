Usage:

    crx-download [url] [output_file] [browser_version]

Examples:

    crx-download 'https://chrome.google.com/webstore/detail/ublock-origin-extra/pgdnlhfefecpicbbihgmbmffkjpaplco'

    crx-download 'https://chrome.google.com/webstore/detail/pgdnlhfefecpicbbihgmbmffkjpaplco'

    crx-download ublock-origin-extra/pgdnlhfefecpicbbihgmbmffkjpaplco

    crx-download pgdnlhfefecpicbbihgmbmffkjpaplco

    crx-download pgdnlhfefecpicbbihgmbmffkjpaplco extension.crx

    crx-download pgdnlhfefecpicbbihgmbmffkjpaplco extension.crx 83.0.4103.97

Note that for downloading files you also need curl, wget or aria2c.
